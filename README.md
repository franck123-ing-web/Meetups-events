# Meetups Events

## Mini-projet React Native

Application mobile permettant de consulter une liste d'événements (meetups), de rechercher un événement et d'afficher les détails.

- React Navigation

## Fonctionnalités

- Affichage d'une liste d'événements
- Recherche d'événements
- Page de détail d'un événement
- Ajout en favori
